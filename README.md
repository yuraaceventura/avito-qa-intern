To run tests:  

Clone the repository: use ```git clone https://github.com/yuraaceventura/avito-qa-intern.git```  
Go to the root directory of the repository ```cd avito-qa-intern```  
Create virtual environment: use ```python3 -m venv .venv```  
Activate virtual environment: use ```source .venv/bin/activate``` on Linux or ```.venv/Scripts/activate``` on Windows  
Install dependencies: use ```pip install -r requirements.txt```  
Being in the root directory of the repository run the tests: use ```pytest -v```  


Для запуска тестов:  

Клонируйте репозиторий: используйте ```git clone https://github.com/yuraaceventura/avito-qa-intern.git```  
Перейдите в корневой каталог репозитория ```cd avito-qa-intern```  
Создайте виртуальное окружение: используйте ```python3 -m venv .venv```  
Активируйте виртуальное окружение: используйте ```source .venv/bin/activate``` для Linux или ```.venv/Scripts/activate``` для Windows  
Установите зависимости: используйте ```pip install -r requirements.txt```  
Находясь в корневом каталоге репозитория, запустите тесты: используйте ```pytest -v```
